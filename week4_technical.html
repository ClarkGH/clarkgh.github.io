Enumerable#cycle

4/03/2014

     The enumerable cycle method is useful when you want to perform a function several times on each of the objects inside of an array.

    

  1. Cycle Pseudocode Syntax:

    2. new_array = []
       new_array.cycle(number_to_cycle) { |array_object| code performed on array_object }

  2. Cycle example:

    3. new_array = %w(bacon, pig, pork, ribs)
       new_array.cycle(2) { |x| print x.capitalize + ", " }
       #=> will cycle through the array twice and end up printing: bacon, pig, pork, ribs bacon, pig, pork, ribs

     Notice how Enumberable#cycle when printing the array objects all twice doesn't put a comma after ribs in the example? If we were to run the same array one time in an each do function we would get the answer: "bacon, pig, pork, ribs"
     In the case of cycling through the array, it takes in the amount of times you want to perform an each do function with simpler and cleaner code.

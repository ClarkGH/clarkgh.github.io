Recursion

     Recursion occurs when a program calls itself directly or indirectly. Recursion is supported by most programming languages where the language allows a function to call itself inside of its text. Recursion allows the ability to define an infinite number of objects and perform an infinite number of computations.

     There are a few functional languages that rely on recursion as they have no constructs defined to loop with. These languages have the ability to solve the same problems without the looping constructs case, while, for, etc..

In order to create a recursive program, there are basic steps that must always be followed:

  1. Begin the initialization of the algorithm.
  2. Ensure that the current value matches the original case.
  3. Redefine the problem into simpler sub categories (refactor).
  4. Test the smaller algorithms.
  5. Combine all of the results of the sub categories and find the answer.
  6. Return the answer/result.

     Below you will see a recursive Ruby function. In this function I use STDIN.gets.chomp(), for a better understanding feel free to click here.

  • Recursion in Ruby:

    • def eat_food(food)
       print food
       eaten = STDIN.gets.chomp
       eaten = eat_food food if eaten.empty?
       return eaten
    end

     The above shows us a recursive function. It calls iteself and will loop again if the input given is empty without using any looping constructs.

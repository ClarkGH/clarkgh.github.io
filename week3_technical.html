Loops in Ruby

3/28/2014

     In programming, there is something called looping, where a block of code is repeated several times until it meets a set amount of requirements. Programmers use these 'loops' to make both theirs and the users' lives easier. Loops not only save a programmer from a great deal of work, they also make the filesize less bulky.

     With ruby, there are several different ways to loop, but they all loop differently. There generally is a loop more appropriate for set situations, and decided what works best depends on what you're looking for.

  1. While

    2.    The While loop will loop continuously as long as the condition given remains true. This is useful when you want your code to loop while the condition you want remains true.

    Pseudocode Syntax:

    while condition do
       code to do
    end

  2. Until

    3.    The until loop will loop continuously as until the condition given remains true. This is best used when you know you want your code to stop looping when it reaches a certain point. This works opposite to while.

    Pseudocode Syntax:

    until condition do
       code to do
    end

  3. For

    4.    The for loop will loop through every item (or element) in a given range. If we know our range, this works best. Most ruby programmers prefer the each do syntax.

    Pseudocode Syntax:

    (expression).each do |element|
       code_to_do
    end

  4. Break

    5.    The Break loop will break out of the most internal loop. This is useful if you have specific parameters you want to end the loop.

    Pseudocode Syntax (with for loop):

    for variable in range given
       if variable meets condition then
       break
    end

  5. Next

    6. The Next loop will move to the next loop. If we want to jump loops this is useful.

    Pseudocode Syntax:

    for variable in range given
       if variable meets condition then
       next
    end

  6. Redo

    7. The Redo loop will restart the loop "iteration" and doesn't check the loop condition. If you want to restart the loop's iteration and not the whole loop when conditions are met, this is the loop to choose.

    Pseudocode Syntax:

    for variable in range given
       if variable meets condition then
       redo
    end

  7. Retry

    8. The retry loop will restart the entire loop over again and not just the iteration. If you want your loop to restart if certain conditions are met, this is what you would want to use.

    Pseudocode Syntax:

    for variable in range given
       if variable meets condition then
       retry
    end
